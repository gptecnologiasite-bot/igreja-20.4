# ADMAC Site PRD

## Overview
ADMAC is a church management and information portal. It provides a public site for church members and visitors, and an admin panel for content management.

## Tech Stack
- React, Vite, Supabase, Lucide-react, React Router.

## Key Features
1. **Public Site**: Home, Ministry pages (Kids, Louvor, Jovens, etc.), About, Contact.
2. **Admin Panel**: Dashboard, Page management, User management, Content management.
3. **Data Persistence**: Supabase for cloud data, localStorage for offline fallback.

## Requirements
- Content updated in the admin panel should reflect on the public site immediately.
- Offline support via localStorage fallback.
- User authentication for admin panel access.
